# Kings-vs.-Goblins – Mobile MVP (1 неделя)

## Core Gameplay
- [x] Вертикальная ориентация для мобильных устройств (Android)
- [x] Автобой юнита игрока (Бомжик) против 1 типа гоблина
- [x] Победа/поражение боя с отображением HP и ATK

## Прогрессия
- [x] Получение ресурса — руда после боя
- [x] Апгрейд королевства (уровень, влияние на добычу)
- [x] Рост сложности врага при апгрейде королевства

## Экран / UI
- [x] Главный экран Королевства
- [x] Кнопка «Вылазка» для начала боя
- [x] Экран лута (руда)
- [x] Кнопка апгрейда королевства и индикатор уровня

## Сохранение / Прогресс
- [x] Локальный Save/Load через JSON / PlayerPrefs
- [x] Сохранение прогресса юнита и уровня королевства

## Графика / Арт
- [x] AI-generated спрайты для юнита игрока
- [x] AI-generated спрайты для врага
- [x] AI-generated иконка ресурса (руда)
- [x] Заглушки для UI
- [x] Взятые Ассеты со стока

## Технологии
- [x] Unity 
- [x] C#
- [x] Canvas UI + TextMeshPro
- [x] GitHub для контроля версий
